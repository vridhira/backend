# Licensing Overview

Quick navigation for the current licensing framework.

## Primary Rule

Commercial rights are not granted by default.
Commercial use requires prior written approval and a signed commercial agreement.

## Core Documents

| Document | Purpose |
|---|---|
| LICENSE | Public non-commercial license text |
| COMMUNITY-LICENSE.md | Community and non-commercial usage conditions |
| COMMERCIAL-LICENSE.md | Public outline of B2B commercial terms |
| COMMERCIAL-LICENSE-AGREEMENT.md | Fill-and-sign commercial contract template |
| PAYMENT-SCHEDULE.md | Invoicing, renewal timing, and non-payment effect |
| LICENSING-GUIDE.md | Decision guidance and commercial application (Appendix A) |
| FORK_LICENSE_RENEWAL_NOTICE.md | Mandatory acknowledgement and compliance warning for fork owners |

## Commercial Request Flow

1. Review LICENSE and COMMERCIAL-LICENSE.md.
2. Submit Appendix A from LICENSING-GUIDE.md.
3. Execute COMMERCIAL-LICENSE-AGREEMENT.md.
4. Complete invoicing per PAYMENT-SCHEDULE.md.
